# AI-Generated Performance Report Card

## Overview
The game now features a comprehensive AI-generated Performance Report Card that appears after completing your cooking session (10 orders). This report card provides detailed analysis of your performance with contextual AI-generated feedback.

## Features

### 1. **Comprehensive Grading System**
- **Overall Grade**: Letter grade (A+ to D) with GPA calculation
- **Speed Grade**: Based on average order completion time
- **Accuracy Grade**: Based on mistakes per order
- **Adaptability Grade**: How well you handle rule changes
- **Pressure Grade**: Stress management under time constraints

### 2. **AI-Generated Sections**
All text sections are dynamically generated by the AI Chef based on your actual performance:

- **Overall Assessment** (40 words): Chef's honest evaluation of your session
- **Strengths** (30 words): 2-3 specific things you did well
- **Weaknesses** (30 words): Areas needing improvement
- **Improvement Tips** (35 words): Actionable advice for next session
- **Chef's Final Notes** (15 words): Memorable closing line from the Chef

### 3. **Detailed Statistics**
- Orders completed vs target
- Final score
- Perfect orders (no mistakes)
- Timeouts
- Fast/slow completions
- Ingredients and tools banned
- Escalation level reached
- Most-used ingredients
- Tool usage patterns

## How It Works

1. **During Gameplay**: The system tracks all your actions:
   - Ingredient selection patterns
   - Tool usage frequency
   - Order completion times
   - Mistakes and their types
   - Stress level changes

2. **AI Analysis**: When you view the report:
   - Statistics are calculated from tracked data
   - Grades are assigned based on performance thresholds
   - AI generates contextual text based on your specific stats
   - Multiple AI prompts are sent to get different perspectives

3. **Fallback System**: If AI is unavailable:
   - Pre-written contextual messages are used
   - Still fully functional with all stats and grades
   - Seamless experience either way
